**Lo que te falta para un ERP grande (sin producción):**

| Módulo | Submódulos clave |
|---|---|
| Recursos Humanos (RRHH) | Empleados, contratos, asistencia, planilla, vacaciones, liquidaciones, evaluaciones de desempeño |
| Nómina / Payroll | Cálculo de sueldos, AFP/ONP, ESSALUD, retenciones 5ta categoría, boletas de pago, PDT 601 |
| Activos Fijos | Registro de activos, depreciación, mantenimiento, bajas y revaluaciones |
| Compras y Proveedores | Solicitudes de compra, cotizaciones de proveedores, órdenes de compra, recepción, evaluación proveedores |
| Logística y Distribución | Guías de remisión, rutas de entrega, seguimiento de envíos, devoluciones |
| Gestión de Proyectos | Proyectos, tareas, Gantt, recursos asignados, avance, costos por proyecto |
| Tesorería | Caja chica, arqueos, posición de caja diaria, conciliación bancaria automatizada |
| Facturación Electrónica | Emisión de facturas/boletas/NC/ND SUNAT, envío OSE, consulta CDR |
| Reportes y BI | Dashboard ejecutivo, KPIs consolidados, reportes regulatorios SUNAT/SBS |
| Configuración / Admin | Usuarios, roles y permisos, empresas, sucursales, parámetros generales |
